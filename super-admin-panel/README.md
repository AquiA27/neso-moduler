# Neso Super Admin Panel

Bu uygulama, Neso SaaS platformu için ayrı bir Super Admin yönetim panelidir. Müşteri uygulamasından tamamen bağımsızdır ve sadece platform yöneticileri tarafından kullanılır.

## Özellikler

- İşletme Yönetimi
- Abonelik Yönetimi
- Ödeme Takibi
- Özelleştirme Yönetimi
- Hızlı İşletme Kurulumu

## Kurulum

```bash
npm install
npm run dev
```

## Çalıştırma

Uygulama `http://localhost:5174` portunda çalışır (müşteri uygulaması 5173'te).

## Giriş

Sadece `super_admin` rolüne sahip kullanıcılar giriş yapabilir.
